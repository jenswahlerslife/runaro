🔄 Complete Integration Flow

1. User klikker "Forbind Strava" på localhost:8080 2. OAuth redirect til Strava med localhost callback 3. Bruger godkender → callback med authorization code 4. strava-auth function udveksler code til tokens 5. Tokens gemmes i database → redirect til success page 6. strava-activities function henter løberuter fra Strava 7. UI viser ruter med "Overfør" knapper 8. transfer-activity function: - Opretter user_activities tabel (hvis den ikke findes) - Gemmer aktivitet med point-beregning - Opdaterer brugers total points - Fjerner aktivitet fra listen
